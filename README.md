Goal Grid is a simple task management app built with Expo. Users can add tasks by typing and pressing the "+" button, and delete tasks by touching them.

## Features
- Add tasks easily
- Remove tasks by touching them
- Minimal and user-friendly UI

## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/ShenanVindinu/goal-grid.git
   cd goal-grid
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the Expo development server:
   ```sh
   npx expo start
   ```

## Usage
- Open the app in an emulator or on your physical device.
- Type a task and press the "+" button to add it.
- Touch a task to delete it.

## Technologies Used
- React Native
- Expo
- TypeScript
